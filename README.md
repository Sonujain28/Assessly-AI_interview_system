# Assessly-AI_interview_system
Assessly is an AI-powered interview preparation system that generates real-time questions for users and evaluates their answers intelligently. It’s designed for students and job aspirants to practice interviews in a simulated, interactive environment.

🚀 #Installation & Setup
Clone the repository:
git clone https://github.com/<Sonujain28>/Assessly-Ai_Interview_system.git
cd Assessly-Ai_Interview_system
Create and activate a virtual environment:
# Linux / Mac
python -m venv venv
source venv/bin/activate
# Windows
python -m venv venv
venv\Scripts\activate 

#Create a .env file in the root folder:
OPENROUTER_API_KEY=your_openai_api_key_here
SECRET_KEY=your_secret_key_here

Run the app locally:
python app.py

Open http://127.0.0.1:5000 in your browser to access the application.
